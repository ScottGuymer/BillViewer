# Tech and design

BillViewer uses the following tech

* Mvc 5
* Castle Windsor
* NLog
* Material Design Lite
* CQRS (the Q part at least)

# How to Run

 - Simply open up BillViewer.sln
 - Restore nuget packages
 - Set BillViewer as start up project
 - Run or Debug!
